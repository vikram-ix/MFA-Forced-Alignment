# Dataset Description

This folder contains the speech data used for the forced alignment assignment.

## Contents

### Audio Files (.wav)
Speech recordings used for alignment experiments.

Examples:
- F2BJRL1.wav
- F2BJRL2.wav
- F2BJRL3.wav
- ISLE_SESS0131_BLOCKD02_01_sprt1.wav
- ISLE_SESS0131_BLOCKD02_02_sprt1.wav
- ISLE_SESS0131_BLOCKD02_03_sprt1.wav

### Annotation Files
- .lab files: Transcriptions corresponding to audio
- .TextGrid files: Praat alignment outputs

## Purpose
These files are used to perform word-level and phoneme-level forced alignment using Praat.
